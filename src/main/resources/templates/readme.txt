# 온라인 벼룩시장 프로젝트

## 소개
**온라인 벼룩시장**은 누구나 회원가입하여 중고 물품을 구매하거나 판매할 수 있는 플랫폼입니다.
사용자 간 실시간 채팅 기능을 제공하여 거래를 더욱 편리하게 만들어줍니다.

---

## 주요 기능
1. **회원 관리**
   - 회원가입 및 로그인

2. **상품 관리**
   - 상품 등록
   - 상품 목록 보기
   - 상품 상세 보기
   - 상품 수정 및 삭제

3. **거래 기능**
   - 구매자와 판매자 간 실시간 채팅

5. **실시간 채팅**
   - 거래 중인 사용자와 실시간 메시지 주고받기

## 기술 스택
- **백엔드**: Spring Boot, Spring Data JPA
- **프론트엔드**: Thymeleaf, HTML/CSS, JavaScript
- **데이터베이스**: MySQL
- **실시간 통신**: WebSocket, STOMP
